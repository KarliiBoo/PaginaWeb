¿Qué es un conjunto?

Un conjunto es una colección o agrupación bien definida de objetos, llamados elementos, que tienen alguna característica en común. Estos objetos pueden ser concretos (como personas o cosas) o abstractos (como números, ideas o relaciones).

Se suele escribir entre llaves, por ejemplo: A = {1, 2, 3}.

Ejemplos rápidos

  • A = {1, 2, 3}
  • B = {3, 4, 5}
  • C = {a, e, i, o, u}

Tip: En conjuntos, no importa el orden y no se repiten elementos.

Tipos de conjuntos

  • Conjunto vacío (∅): No tiene elementos. Ejemplo: ∅ = { }.
  • Conjunto finito: Tiene un número limitado de elementos. Ejemplo: A = {1, 2, 3}.
  • Conjunto infinito: Tiene un número ilimitado de elementos. Ejemplo: Números naturales = {0, 1, 2, 3, ...}.
  • Subconjunto (⊆): Todos los elementos de un conjunto están en otro. Ejemplo: Si A = {1, 2} y B = {1, 2, 3}, entonces A ⊆ B.
  • Conjunto universal (U): Contiene todos los elementos posibles en un contexto dado.

Propiedades de las operaciones con conjuntos

  • Conmutativa: A ∪ B = B ∪ A y A ∩ B = B ∩ A
  • Asociativa: (A ∪ B) ∪ C = A ∪ (B ∪ C) y (A ∩ B) ∩ C = A ∩ (B ∩ C)
  • Distributiva: A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C) y A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
  • Absorción: A ∪ (A ∩ B) = A y A ∩ (A ∪ B) = A
  • Identidad: A ∪ ∅ = A y A ∩ U = A
  • Complemento: A ∪ A' = U y A ∩ A' = ∅
  • Leyes de De Morgan: (A ∪ B)' = A' ∩ B' ; (A ∩ B)' = A' ∪ B' y A - B = A ∩ B'
  • Idempotente: A ∪ A = A y A ∩ A = A
